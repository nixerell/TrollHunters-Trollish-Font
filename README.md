# TrollHunters Trollish Font
Custom font ttf of the Trollish language from the show TrollHunters. Can be used in css, and added to update your system's font. It's entirely accurate to the show/art book translation. 1:1 Trollish to English tranlation so its easy to learn.

# Font Download
[TTF Here](https://file.garden/Zrb-HByrrgxh44dz/Trollish_Trollhunters-Regular.ttf)

# Add To Discord
Download [Better Discord Here](https://github.com/BetterDiscord/BetterDiscord) and open custom CSS.
In the css editor enter:
```
@font-face {
font-family: Trollish_Trollhunters-Regular; 
  src: url(https://file.garden/Zrb-HByrrgxh44dz/Trollish_Trollhunters-Regular.ttf); 
}

p {
  font-family: Trollish_Trollhunters-Regular;
```
and add whatever else you want to your css before clicking save.
